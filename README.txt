🚀 HOW TO OPEN AND USE
================================================================================

1. SAVE ALL FILES IN THE SAME FOLDER
   → This ensures links between files work properly

2. OPEN IN WEB BROWSER
   → Double-click any .html file
   → Recommended: Google Chrome, Microsoft Edge, or Firefox

3. INTERACTIVE CHECKLIST
   → Click any checkbox to mark as complete
   → Checked items show a checkmark

4. PRINT TO PDF
   → Press Ctrl + P (Windows) or Cmd + P (Mac)
   → Choose "Save as PDF" as destination
   → Print and use as needed
